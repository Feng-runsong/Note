# 机器学习与深度学习核心概念笔记

------

## 1. 机器学习与人工智能的关系

- **人工智能**：是最宽泛的概念，旨在让机器模仿人类的智能行为，例如学习、推理、感知等。它是一个宏大的目标。
- **机器学习**：是实现人工智能的一种核心方法。它不依赖于显式编程的规则，而是让计算机利用数据进行学习和改进，从而完成任务。
- **深度学习**：是机器学习的一个子集。它使用包含多个隐藏层的人工神经网络，能够自动从海量数据中提取复杂的特征，在处理非结构化数据（如图像、音频、文本）方面表现优异。

**关系总结**：人工智能包含机器学习，机器学习包含深度学习。深度学习是机器学习的一种，而机器学习是实现人工智能的主要途径。

------

## 2. 神经网络基础

### 2.1 神经网络的基本构成

一个基础的神经网络由三个主要部分组成：

1. **输入层**：
   - 负责接收原始数据。神经元的个数通常与特征的维度（即有多少个输入变量）相同。
   - 这一层不进行计算，只是将数据传递到下一层。
2. **隐藏层**：
   - 位于输入层和输出层之间，是网络的核心计算部分。
   - 可以有一层或多层。层数越多，网络越"深"，能学习的特征也越抽象、复杂。
   - 每个神经元接收上一层的输入，进行加权求和，再通过一个激活函数，将结果传递给下一层。
3. **输出层**：
   - 负责输出最终的结果。神经元的个数取决于任务类型。
   - 例如：二分类任务通常为1个神经元；多分类任务则为多个神经元。

### 2.2 神经元——网络的基本单元

- **概念**：神经元是神经网络中最基本的计算单元，灵感来源于生物神经元。
- **内部结构**：一个神经元接收多个输入，每个输入对应一个权重，加上一个偏置项。神经元的工作分为两步：
  1. **加权求和**：将输入与对应权重相乘后全部相加，再加上偏置。
  2. **激活函数**：将加权求和的结果输入到激活函数中，得到最终的输出。

------

## 3. 神经网络的工作流程

### 3.1 前向传播——如何预测

- **定义**：数据从输入层开始，经过一层又一层的计算（加权求和→激活），最终传递到输出层，产生预测结果的过程。信息是单向"向前"流动的。
- **过程**：
  1. 将特征向量送入输入层。
  2. 在第一个隐藏层，每个神经元对输入进行加权求和并应用激活函数，产生该层的输出。
  3. 将第一隐藏层的输出作为输入，传递给第二隐藏层，重复计算过程。
  4. 直到输出层，产生最终的预测值。
- **本质**：前向传播就是网络在给定当前权重下，进行"推理"或"预测"的过程。

### 3.2 反向传播——如何学习与训练

- **定义**：是训练神经网络的核心算法。它通过计算损失函数对网络权重的梯度，并将该梯度从输出层向输入层"反向"传播，从而指导权重的更新。
- **"学习"与"训练"过程**：
  1. **前向传播**：输入一批数据，得到预测值。
  2. **计算损失**：通过损失函数计算预测值与真实值之间的差距（即误差）。
  3. **反向传播误差**：利用链式法则，从输出层开始，逐层反向计算损失函数对每个权重的偏导数（即梯度）。这个梯度指明了该权重对最终误差的"贡献"大小和方向。
  4. **更新权重**：使用梯度下降等优化算法，沿着梯度的反方向（即减小误差的方向）微调每个权重。
  5. **迭代**：重复以上步骤，直到损失函数收敛到一个较小的值。此时，我们说网络"训练"好了，"学习"到了数据中的模式。

## 4. 关键概念与原理

### 4.1 激活函数

- **是什么**：是神经元中的非线性函数。如果没有激活函数，无论神经网络有多少层，其输出都是输入的线性组合，无法解决复杂问题。激活函数引入非线性，使网络能够学习复杂的模式。
- **主流激活函数**：
  - **Sigmoid**：将输入映射到0和1之间，常用于二分类输出层。缺点是容易导致梯度消失，计算量大。
  - **Tanh（双曲正切）**：将输入映射到-1和1之间，输出均值为0，效果常优于Sigmoid。同样存在梯度消失问题。
  - **ReLU（修正线性单元）**：目前最常用的隐藏层激活函数。计算简单，能缓解梯度消失。缺点是部分神经元可能"坏死"（永远不被激活）。

### 4.2 损失函数

- **是什么**：用来衡量模型预测值与真实值之间不一致程度的函数。损失值越小，模型性能越好。它是模型优化的目标。
- **常见损失函数**：
  - **均方误差**：用于回归问题。计算预测值与真实值差的平方的平均值。
  - **交叉熵损失**：用于分类问题。衡量两个概率分布之间的差异。

### 4.3 数据集划分

为了客观评估模型，通常将数据划分为三个互不相交的集合：

1. **训练集**：
   - **用途**：用于训练模型，即让模型"学习"数据中的模式，通过反向传播更新权重。
   - **占比**：通常最大，约占60%到80%。
2. **验证集**：
   - **用途**：用于模型选择和调参。在训练过程中，用它来评估不同模型架构或超参数的性能，防止过拟合。模型不直接在验证集上学习更新权重。
   - **占比**：约占10%到20%。
3. **测试集**：
   - **用途**：用于最终评估已训练好的模型的泛化能力。仅在模型全部确定后使用一次，模拟模型在真实世界新数据上的表现。
   - **占比**：约占10%到20%。

## 5. 机器学习深入概念

### 5.1 学习范式

- **监督学习**：训练数据包含输入和标签。模型学习从输入到标签的映射。例如：分类、回归。
- **无监督学习**：训练数据只有输入，没有标签。模型自行发现数据中的隐藏结构或模式。例如：聚类、降维。
- **半监督学习**：结合少量有标签数据和大量无标签数据进行训练，常用于标签获取成本高昂的场景。

### 5.2 过拟合

- **是什么**：模型在训练数据上表现过好，以至于"死记硬背"了数据中的噪声和异常值，但在未见过的测试数据上表现很差。即泛化能力弱。
- **为什么会过拟合**：
  1. 模型过于复杂（参数太多）。
  2. 训练数据太少。
  3. 数据噪声太多。
  4. 训练时间过长。
- **如何判断**：训练误差持续下降，但验证误差开始上升。

### 5.3 防止过拟合的策略

- **正则化**：在损失函数中添加一个惩罚项，限制模型的复杂度。
  - **L1正则化**：惩罚权重的绝对值之和，可以产生稀疏权重。
  - **L2正则化**：惩罚权重的平方和，使权重尽可能小且分散。
- **数据归一化**：
  - **方法**：将数据按比例缩放到一个小的特定区间，如0到1之间，或使其均值为0、标准差为1。
    - **Min-Max归一化**：用当前值减去最小值，再除以最大值减最小值的差。
    - **Z-core标准化**：用当前值减去均值，再除以标准差。
  - **作用**：1）消除不同特征之间的量纲影响，使模型更稳定。2）加速梯度下降的收敛速度。
- **数据清洗**：在训练前，对原始数据进行处理，包括处理缺失值、去除重复数据、修正错误数据、处理异常值等。这是保证模型效果的基础。
- **早停**：在训练过程中，当验证集上的性能不再提升甚至下降时，提前停止训练。

### 5.4 训练中的关键参数

- **批大小**：每次训练（一次前向传播加反向传播）送入网络的样本数量。
  - **太大**：训练稳定，但内存消耗大，可能陷入局部最优。
  - **太小**：收敛不稳定，但可能跳出局部最优。
- **学习率**：控制权重更新步长大小的超参数。
  - **太大**：模型可能在最优解附近震荡，无法收敛，甚至发散。
  - **太小**：收敛速度极慢，可能陷入局部最优。

## 6. 经典算法详解

### 6.1 线性回归

目标是找到一条直线（或超平面）来最佳拟合数据点。

- **最小二乘法构建损失函数**：最常用的方法是使预测值与真实值的残差平方和最小。这就是均方误差损失函数的来源。
- **解析求解**：通过数学公式直接一步计算出最优权重。适用于特征数量不多的小数据集。
- **梯度下降法求解**：一种迭代优化算法，广泛适用于大数据集和复杂模型。
  1. 随机初始化权重。
  2. 计算损失函数对权重的梯度（即偏导数）。
  3. 沿着梯度的反方向更新权重。
  4. 重复步骤2和3，直到收敛。

### 6.2 K-means 聚类算法

一种经典的无监督学习算法，用于将数据划分为K个簇。

- **核心思想**：以距离（如欧氏距离）作为相似性度量，将离得近的样本聚在一起。
- **算法步骤**：
  1. **初始化**：随机选择K个样本点作为初始簇中心。
  2. **分配**：对于数据集中的每个样本，计算它到K个簇中心的距离，并将其分配到距离最近的簇中心所在的簇。
  3. **更新**：对于每个簇，计算该簇内所有样本的平均值，作为新的簇中心。
  4. **迭代**：重复步骤2和3，直到簇中心的变化很小，或达到最大迭代次数。
- **特点**：简单高效，但需要预先指定K值，且对初始中心点的选择敏感。
